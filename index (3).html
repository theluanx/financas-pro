🏦 Configurar Pluggy — Open Finance

1
Crie sua conta no Pluggy (grátis)
Acesse o dashboard do Pluggy, crie um novo app e copie seu Client ID e Client Secret.
→ Abrir dashboard.pluggy.ai
2
Deploy do backend na Vercel (grátis)
O Client Secret não pode ficar exposto no HTML. Crie um projeto na Vercel com este arquivo:
// api/pluggy-token.js export default async function handler(req, res) { res.setHeader('Access-Control-Allow-Origin', '*'); const auth = await fetch('https://api.pluggy.ai/auth', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ clientId: process.env.PLUGGY_CLIENT_ID, clientSecret: process.env.PLUGGY_CLIENT_SECRET }) }); const { apiKey } = await auth.json(); const token = await fetch('https://api.pluggy.ai/connect_token', { method: 'POST', headers: { 'X-API-KEY': apiKey, 'Content-Type': 'application/json' }, body: JSON.stringify({}) }); const { accessToken } = await token.json(); res.json({ connectToken: accessToken }); }
⚙️ Nas configurações da Vercel, adicione as variáveis de ambiente:
PLUGGY_CLIENT_ID e PLUGGY_CLIENT_SECRET
